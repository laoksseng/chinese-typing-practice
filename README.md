# 中文打字練習網頁

一個純靜態的中文打字練習工具，支援計時練習、速度統計、準確率統計、文章進度提示，以及自定義練習文章導入。

## 功能特色

- 中文打字練習，適合繁體中文文章
- 可選擇 `5 分鐘`、`10 分鐘`、`20 分鐘`
- 支援自定義練習時間，單位為分鐘
- 即時顯示：
  - 剩餘時間
  - 打字速度，單位為字/分
  - 準確率
  - 文章完成進度
- 速度、準確率、進度均有視覺化進度條
- 練習輸入框禁止貼上及拖放文字
- 支援漏字對齊，避免漏打一字後整段被判錯
- 支援三種文章導入方式：
  - 直接貼上文字
  - 導入 `.txt` 檔案
  - 輸入網站網址，自動嘗試提取正文

## 預設文章

預設文章為澳門基本法相關中文文本，可直接開始練習。也可以使用右側「文章導入」模塊替換為自己的文章。

## 使用方式

直接用瀏覽器打開 `index.html` 即可使用。

如果希望用本機伺服器預覽，可以在專案資料夾執行：

```bash
python -m http.server 5173 --bind 127.0.0.1
```

然後打開：

```text
http://127.0.0.1:5173/
```

## 文章導入說明

### 直接貼上文字

在「貼上文字」分頁中貼上文章，點擊「使用此文章」即可。

### 導入 TXT 檔案

在「TXT 檔案」分頁選擇 `.txt` 純文字檔。建議使用 UTF-8 編碼。

### 導入網站文章

在「網站文章」分頁輸入文章網址，系統會嘗試自動提取標題與正文。

目前已針對澳門日報電子版文章做了特別處理，例如：

```text
https://www.macaodaily.com/html/2026-06/29/content_1917564.htm
```

注意：網站文章提取會受到網站跨域政策、網頁結構、代理服務穩定性等因素影響。若提取失敗，建議直接複製文章正文到「貼上文字」分頁。

## 部署

本專案是純靜態網頁，不需要後端或資料庫，可部署到：

- GitHub Pages
- Netlify
- Vercel
- 任何靜態網站主機

### GitHub Pages 部署建議

1. 建立 GitHub repository
2. 上傳 `index.html`、`style.css`、`app.js`、`README.md`
3. 到 repository 的 `Settings`
4. 進入 `Pages`
5. Source 選擇要發佈的 branch，例如 `main`
6. 資料夾選擇 `/root`
7. 儲存後等待 GitHub Pages 產生網址

## 技術說明

- 使用原生 HTML、CSS、JavaScript
- 不依賴前端框架
- 不需要安裝套件
- 所有資料只在瀏覽器端處理，不會主動保存使用者輸入內容

## 已知限制

- 網站文章提取不保證所有網站都成功
- 某些網站可能阻擋跨域讀取或代理讀取
- 中文輸入法組字期間會等組字完成後再更新判斷
- 打字速度以正確輸入字數計算，單位為字/分
